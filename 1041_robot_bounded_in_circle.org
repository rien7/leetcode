#+title: 1041. Robot Bounded In Circle
#+filetags: :Medium:

* 问题
On an infinite plane, a robot initially stands at ~(0, 0)~ and faces north. The robot can receive one of three instructions:

- ~"G"~: go straight 1 unit;
- ~"L"~: turn 90 degrees to the left;
- ~"R"~: turn 90 degrees to the right.

The robot performs the ~instructions~ given in order, and repeats them forever.

Return ~true~ if and only if there exists a circle in the plane such that the robot never leaves the circle.

*Example 1:*

#+BEGIN_SRC
Input: instructions = "GGLLGG"
Output: true
Explanation: The robot moves from (0,0) to (0,2), turns 180 degrees, and then returns to (0,0).
When repeating these instructions, the robot remains in the circle of radius 2 centered at the origin.
#+END_SRC

Example 2:

#+BEGIN_SRC
Input: instructions = "GG"
Output: false
Explanation: The robot moves north indefinitely.
#+END_SRC

Example 3:

#+BEGIN_SRC
Input: instructions = "GL"
Output: true
Explanation: The robot moves from (0, 0) -&gt; (0, 1) -&gt; (-1, 1) -&gt; (-1, 0) -&gt; (0, 0) -&gt; ...
#+END_SRC

*Constraints:*

- ~1 <= instructions.length <= 100~
- ~instructions[i]~ is ~'G'~, ~'L'~ or, ~'R'~.

* 思路

** 思路1
本题并没有用到什么算法或数据结构, 主要考察的是对题目的理解.

对于坐标运动的题目来说, 用 *二维数组代表四个方向* 以及 *x, y 坐标来代表当前位置* 是基本操作

然后就是对循环以及转圈的理解, 对于无限循环的运动指令来说, 要想成环, 有两种情况.

要么回到原点(1个循环成环), 要么一组指令最后的方向不为前进方向(多个循环成环)

** 思路2
在以下的代码中, =cur= 代表的就是当前的方向, =0= 代表前进, =1= 代表左转, =2= 代表后退, =3= 代表右转. 与方向坐标 =dirs= 一一对应

* 代码
#+BEGIN_SRC go
func isRobotBounded(instructions string) bool {
    dirs := [][]int{{1, 0}, {0, -1}, {-1, 0}, {0, 1}}
    cur, x, y := 0, 0, 0
    for _, s := range instructions {
        switch s {
        case 'L':
            cur = (cur + 1) % 4
        case 'R':
            cur = (cur + 3) % 4
        case 'G':
            x += dirs[cur][0]
            y += dirs[cur][1]
        }
    }
    return x == 0 && y == 0 || cur != 0
}
#+END_SRC
